# AttnGAN Enhanced Pipeline ‚Äî Text-to-Image Research System

Generate photorealistic bird images from text descriptions using the pretrained
**AttnGAN** model (Xu et al., 2018), with a three-stage enhancement pipeline
for measurably superior output quality.

## Quick start

### Baseline (original AttnGAN)

```bash
python main_kaggle.py
```

Outputs 256√ó256 PNG images to `outputs/`.

### Enhanced pipeline (single run)

```bash
python main_enhanced.py
```

Runs the three-stage enhanced pipeline (CLIP-reranked, 512√ó512 output)
and saves to `outputs/enhanced/`.

### Full comparison experiment

```bash
python run_experiment.py
```

### Train BERT text encoder (new)

```bash
python train_bert_attngan.py
```

This creates BERT-compatible weights in `data/` by default:

- `bert_text_encoder.pth`
- `bert_AttnGAN_generator.pth`

The training pipeline keeps `G_NET` architecture unchanged and distills
`bert-base-uncased` text features into the original AttnGAN text embedding
space (`cfg.TEXT.EMBEDDING_DIM`, default 256), so inference remains compatible.

Runs **both** baseline and enhanced pipelines on 12 standardised prompts,
scores all outputs with CLIP, and generates:

- `outputs/baseline/`          ‚Äî raw AttnGAN 256√ó256 images
- `outputs/enhanced/`          ‚Äî enhanced 512√ó512 images
- `outputs/comparison_grid.png` ‚Äî side-by-side visual comparison
- `outputs/comparison_report.md` ‚Äî research-style markdown report

## Enhancement Details

### 1. CLIP-Guided Generation

`src/clip_scorer.py` wraps CLIP ViT-B/32 with two backend options:

| Backend | Install | Notes |
|---------|---------|-------|
| `transformers` | `pip install transformers` | Recommended; pre-installed on most Kaggle kernels |
| `openai-clip` | `pip install openai-clip` | Original OpenAI package |
| heuristic | (none) | Laplacian sharpness proxy when no CLIP is available |

```python
from src.clip_scorer import CLIPScorer

scorer = CLIPScorer()
score  = scorer.score(pil_image, "a red bird with blue wings")
best, scores, idx = scorer.rerank([img1, img2, img3], text)
```

### 2. Attention Refinement

`src/attention_refinement.py` introduces `RefinedGlobalAttentionGeneral`:

- **Temperature scaling** ‚Äî divides attention logits by œÑ = 0.7 before softmax,
  producing sharper word-region alignment without any new trainable parameters.
- **Top-k word masking** ‚Äî suppresses the bottom 25% of word attention scores
  per query position, focusing the generator on content words.

The `RefinedG_NET` class loads the original `bird_AttnGAN2.pth` checkpoint
**without modification** (identical state-dict key structure).

### 3. Multi-Stage Post-Processing

`src/enhancer.py` applies a 5-step enhancement chain:

1. Lanczos upscale 256 ‚Üí 512 px
2. Gaussian denoising (œÉ = 0.6) ‚Äî suppresses GAN checkerboard artefacts
3. UnsharpMask sharpening ‚Äî recovers fine structural detail
4. Contrast boost √ó1.15
5. Saturation boost √ó1.08

Optionally uses Real-ESRGAN (4√ó learned super-resolution) if `realesrgan`
and `basicsr` are installed.

## Kaggle Quick Start

### Step 1 ‚Äî Upload pretrained weights

Create a Kaggle Dataset (e.g. `attngan-pretrained`) containing:
- `captions.pickle`
- `text_encoder200.pth`
- `bird_AttnGAN2.pth`

If you trained BERT weights, also include:
- `bert_text_encoder.pth`
- `bert_AttnGAN_generator.pth`

### Step 2 ‚Äî Configure paths

In `run_experiment.py` (or `main_enhanced.py`), set:

```python
MODEL_DIR   = "/kaggle/input/attngan-pretrained"
OUTPUT_ROOT = "/kaggle/working/outputs"
```

For BERT inference:

```python
MODEL_DIR = "/kaggle/input/your-model-folder"
cfg.TRAIN.NET_E = "bert_text_encoder.pth"
cfg.TRAIN.NET_G = "bert_AttnGAN_generator.pth"
cfg.TEXT.ENCODER_TYPE = "bert"
```

### Step 3 ‚Äî Install CLIP (internet-enabled session)

```bash
pip install transformers ftfy regex
```

### Step 4 ‚Äî Run

```python
%run AttnGANEnhancement/run_experiment.py
```

## Configuration reference

| Parameter | Default | Location | Effect |
|-----------|---------|----------|--------|
| `NUM_CANDIDATES` | 6 | `run_experiment.py` | CLIP-ranked candidates per prompt |
| `ATTN_TEMP` | 0.7 | `run_experiment.py` | Attention temperature œÑ (lower = sharper) |
| `TOP_K_RATIO` | 0.75 | `run_experiment.py` | Fraction of words retained by top-k mask |
| `BASELINE_SEED` | 42 | `run_experiment.py` | Fixed seed for baseline reproducibility |
| `cfg.GAN.GF_DIM` | 32 | `config/kaggle_config.py` | Generator feature-map base width |
| `cfg.TEXT.EMBEDDING_DIM` | 256 | `config/kaggle_config.py` | Text encoder output dim |

## Python API

```python
import sys
sys.path.insert(0, "path/to/AttnGANEnhancement")

from config.kaggle_config import cfg
cfg.TRAIN.NET_E = "text_encoder200.pth"
cfg.TRAIN.NET_G = "bird_AttnGAN2.pth"

# Baseline
from src.model_wrapper import AttnGANWrapper
baseline = AttnGANWrapper("data/")
img = baseline.generate("a small red bird with blue wings")
img.save("baseline.png")

# Enhanced
from src.enhanced_pipeline import EnhancedAttnGANWrapper
enhanced = EnhancedAttnGANWrapper("data/", num_candidates=6, temperature=0.7)
img, clip_score, all_scores = enhanced.generate_with_scores("a small red bird with blue wings")
img.save("enhanced.png")  # 512√ó512 PNG
print(f"CLIP score: {clip_score:.3f}")
```
